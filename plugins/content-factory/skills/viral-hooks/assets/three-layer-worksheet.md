# Three-layer hook worksheet

Fill this in before shooting. The point is to pressure-test that the
visual, verbal, audio, and on-screen text layers all point at the same
idea. A misaligned three-layer hook reads as confused; an aligned one
reads as inevitable.

For the longer explanation, see `references/three-layer-hook.md`.

```
VIDEO IDEA:
ARCHETYPE:           (see references/hook-archetypes.md)
PLATFORM:            (TikTok / Reels / Shorts)
OBJECTIVE:           (reach / retention / authority / conversion)
PROOF SOURCE:

────────────────────────────────────────────────────────────────────────
LAYER          | CONTENT                              | REVIEW POINT
────────────────────────────────────────────────────────────────────────
VISUAL         | [WHAT THE FIRST FRAME SHOWS]         | paused frame
               | [WHY IT WORKS PAUSED]                |
────────────────────────────────────────────────────────────────────────
VERBAL         | [NATURAL SPOKEN LINE]                 | early
               | [POWER VERB? CONCRETE NOUN?]         |
────────────────────────────────────────────────────────────────────────
ON-SCREEN TEXT | [GLANCEABLE, BIG, READABLE]           | opening beat
               | [COMPRESSES VERBAL, DOESN'T REPEAT]  |
────────────────────────────────────────────────────────────────────────
AUDIO          | [VOICE / MUSIC / SILENCE IF USEFUL]  | aligned beat
               | [WHAT IT TIMES TO]                   |
────────────────────────────────────────────────────────────────────────
```

## Pressure tests

Before shooting, run all four:

- [ ] **Muted test.** Cover the audio. Does the first frame plus the
      on-screen text deliver the hook? If no, the visual or text is
      doing too little.
- [ ] **Eyes-away test.** Cover the visual. Does the spoken line pull
      a glancing viewer back? If no, the verbal hook is too soft.
- [ ] **Glance test.** View at phone scale briefly. Did text register? If not,
      shorten, enlarge, or improve contrast.
- [ ] **Alignment test.** Do all three layers point at the same idea,
      or are they three different hooks competing? If competing, pick
      the strongest and align the other two to it.

## Common misalignments

- **High-energy verbal, static visual.** The voice promises a payoff
  the eye can't see. Fix the frame.
- **Strong visual, generic voice.** The eye is sold; the voice doesn't
  carry the promise forward. Sharpen the verbal line.
- **Text repeats the voice word-for-word.** Wastes the layer. Compress
  the text or change what it says.
- **Audio fights the cut.** A beat drop on the wrong frame reads as
  unintentional. Re-time or remove.

## After shooting

Watch opening beat back without context. Ask:

- Is relevance or stake clear quickly enough for this format?
- Does first frame give right viewer a reason to continue?
- If two viewers watched (one with sound, one muted), would both get
  the hook?

If any answer is "no," re-shoot the opener before publishing.
