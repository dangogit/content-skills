# Hashtag Reality

Hashtags label topics and support search. They are not guaranteed reach multipliers. Platform limits and recommendations change, so verify current official documentation before treating a number as policy.

## Instagram

- Use only relevant hashtags.
- If configured project design lessons set a working cap, follow it. Otherwise use
  only relevant tags and verify current platform limits.
- Do not call three to five an algorithmic sweet spot.
- Caption topic words, content quality, and audience response matter more than tag volume.
- Put Latin hashtags on separate lines after Hebrew copy to protect RTL.

## TikTok

- Use relevant topical labels only.
- Avoid unrelated trend tags and generic `#fyp` filler.
- Let actual search traffic and post analytics decide whether tags help.
- Verify current caption and hashtag behavior in TikTok before production handoff.

## YouTube

- Hashtags may appear in title or description, but descriptive title matters more.
- Verify current Shorts classification and hashtag policy before production
  handoff; do not depend on a tag for classification.
- Avoid unrelated or misleading hashtags because they can trigger metadata enforcement.

Official references:

- YouTube hashtag policy: `https://support.google.com/youtube/answer/6390658`
- YouTube Shorts classification: `https://support.google.com/youtube/answer/15424877`

## Decision Rule

Use smallest relevant set. Record actual result before turning a count into house style.
