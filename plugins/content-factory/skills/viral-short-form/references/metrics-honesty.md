# Objective-Aware Metrics

No single metric defines good content. Choose objective before production.

## Reach

- qualified reach;
- views;
- three-second view rate or viewed-vs-swiped when available.

## Retention

- average watch time;
- average watch time divided by video duration;
- completion and rewatch when available.

## Authority

- saves per reach;
- shares per reach;
- useful comments and DMs;
- profile or follower lift when available.

## Conversion

- qualified keyword comments per reach;
- successful DM deliveries;
- guide opens or downstream action when measurable.

Views are not automatically vanity. Saves are not automatically best. A reach post,
proof post, and lead-magnet post serve different jobs.

## Honest Claims

- Never promise virality.
- Never fabricate proof or precision.
- Treat platform mechanics as dated hypotheses unless official docs confirm them.
- Compare against the creator's own same-format baseline.
- Capture 24-hour and 7-day snapshots before promoting a lesson.

## Experiment Rule

Change one main variable per test. Record hook type, story role, format, duration,
CTA type, named artifact, and visual treatment. A result with several changed
variables cannot prove which change worked.
